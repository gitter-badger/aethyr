faq

frequent

Do both my computer and the one I'm downloading from need to have this installed?
No, only the machine that is doing the downloading needs to install this.


There are no libraries for me to select! What's going on?
You need to first connect to the libraries within iTunes before they'll show up in Aethyr.


I've downloaded the music, so where are my files?
You can use the home icon to specify a download location. By default, they're in "My Documents/Aethyr", sorted by their tagged Artist/Album/Title. The download location you specify will be saved, so you won't have to enter it next time. (This also should carry over between updates.) Tip: Set it to "My Music/iTunes/iTunes Media/Automatically Add to iTunes" to do just that.


Aethyr is downloading the files, but they all show up with a filesize of 0 kb.
This should be patched in the latest version, please use the feedback form if otherwise.


This is great! But I have a Mac. When are you going to port it?
Development is halted on this. I have no easy access to a Mac, and virtualization attempts are laggy and crash-prone.


Why do you have so many damn updates?
I'm the only person working on this, and therefore also the only tester with a debugging version of the code. So I'm prone to missing a few crucial bugs here and there.



installation

I just installed Aethyr and it's not starting up, with no error message.
This happens on some computers. Close both iTunes and Aethyr, then start up Aethyr again.


Same as above but a window pops up saying, "Windows cannot find 'aethyrHelper.exe'. Make sure you typed the name correctly, and then try again."
Adobe AIR didn't install properly, most likely because you ran the installer while offline. Go to http://get.adobe.com/air for the latest version.


Aethyr doesn't start up at all, and I've tried everything.
1) You are using an unsupported version of iTunes. You'll need to upgrade.
2) You don't have VC++ installed. Do a quick search online for "Microsoft Visual C++ 2010 Redistributable Package," and install it.
3) Try running the program as administrator.
If none of the above fixes work, then your antivirus/firewall/UAC is most likely blocking Aethyr. Try creating an exception for it.



use

I don't know how to download songs / I can only download one at a time?!
Select songs by clicking on them, use <shift> and <ctrl> to select multiple songs at once. Hit the white arrow to download.


I specified a location using the home icon, but Aethyr will not write there or is writing to 'My Documents/Aethyr' instead.
The cause of this is UAC, which doesn't allow it to write to system-protected directories (this includes 'Program Files').


I can't use iTunes to listen to music while using Aethyr!
I realize that this is a current limitation. If possible, it'll be patched in an upcoming release. Even if it's not possible, the length of time will be reduced from the duration of the download to a few seconds.


Can this search using characters from other languages? (e.g. Chinese, Japanese, Korean)
Yes! You may have to experiment a bit to figure out which keyboard input you need to use. For a starting reference, Chinese requires you to use the "New Experience Input Style" instead of the "ABC Input Style." A guess from this behavior is that you need to use input that displays a horizontal list as opposed to a vertical list.



general

Why do libraries have loading time?
This is a built-in limitation of iTunes that only occurs on the first load of a library. Switching between already loaded libraries is instantaneous.


Can't you combine all the libraries into one super-library?
This isn't possible due to performance issues. Even in iTunes, it takes longer to connect to bigger libraries.


Aethyr seems to stop working in the middle of downloading a set of songs.
This is due to a very rare disconnection bug.


Where do you plan to go from here?
First fix all the bugs that keep cropping up. Afterwards add a 'settings' button to choose which tags you want displayed, allow downloads from multiple libraries at the same time, playback while downloading, allow transfer of other types of media such as pictures and videos, and ripping from local iDevices plugged into your computer.
